PrepMate – AI Study Assistant

Tagline: Your AI-powered university study companion

Overview:
PrepMate helps university students prepare for exams by:

Generating adaptive quizzes (5 questions per session)

Tracking weak topics and providing personalized study recommendations

Creating a study schedule table

Motivating students with daily streak tracking and encouraging messages

Features:

Student Info Page: Collects name, subject, exam date, and daily study time

AI Quiz Page: 5 university-level questions, one at a time

Math: Calculus, Linear Algebra, Probability, Discrete Math

IT/CS: Algorithms, Data Structures, Logic, Databases

Tracks correct answers and weak topics

Results Page: Summarizes performance, highlights weak topics, gives supportive feedback

Study Schedule Table:

Displays Date, Subject, Topic, Study Duration, Days Remaining

Prioritizes weak topics

Allows regeneration

Study Streak Page:

Tracks consecutive study days

Motivates users with encouraging messages

Technology / Platforms:

Airia: Handles quiz logic, weak-topic tracking, and AI responses

Lovable: No-code platform used to create interactive UI and navigation

GitHub: Repository for all project files and documentation

UI Design:

Minimalistic, modern, student-friendly

Light colors: blue, purple, white

Rounded, “bubble-style” buttons

Mobile responsive

How to Use / Test:

The app is built on Lovable (no-code platform) → interface is fully interactive

For AI logic, see airia_prompt.txt file

Study schedule and streak features are linked to quiz completion

Next Steps / Future Improvements:

Expand question bank
